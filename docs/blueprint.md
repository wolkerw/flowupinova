# **App Name**: FlowUp Chat

## Core Features:

- Interactive Chat Interface: Enable users to send and receive messages in a real-time chat format, enhancing user engagement.
- AI-Powered Marketing Assistance: Use an AI tool to provide marketing suggestions, content creation, and data analysis based on user prompts.
- Dynamic Chat Bubbles: Visually distinct chat bubbles for users and AI, improving readability and user experience.
- Error Handling and Notifications: Display error messages within the chat interface to inform users of any issues during AI processing.
- Markdown Support: Render AI responses using Markdown for richer formatting and presentation.
- Animated Loading Indicator: Show a loading indicator when the AI is processing, providing visual feedback to the user.
- Dashboard Navigation: Include a link to navigate to the main dashboard, providing access to other functionalities.

## Style Guidelines:

- Primary color: Sky blue (#7DD3FC), evoking trust and forward movement.
- Background color: Light blue (#F0F9FF), providing a clean and calming backdrop.
- Accent color: Indigo (#3B82F6), for highlights and interactive elements.
- Font: 'Inter', a sans-serif font, for both headlines and body text.
- Use simple, clear icons from Lucide, related to marketing and AI.
- Centered layout with a maximum width for comfortable readability.
- Subtle animations using Framer Motion for smooth transitions and feedback.
